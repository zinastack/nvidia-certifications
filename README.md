# NVIDIA NCP-AII Professional Certification Workspace

This repository is a personal study workspace for the NVIDIA-Certified
Professional: AI Infrastructure (NCP-AII) certification.

Use it to organize official-reference notes, hands-on labs, troubleshooting
practice, and review material.

## Repository Structure

- `docs/` - Exam objective summaries, reference links, and architecture notes.
- `labs/` - Hands-on practice exercises with setup steps, commands, and observations.
- `study-notes/` - Topic-by-topic notes written in your own words.
- `study-notes/templates/` - Reusable note templates.

## Suggested Study Flow

1. Confirm the latest official NCP-AII Professional exam guide.
2. Map each exam blueprint domain to notes and labs.
3. Build hands-on labs for server bring-up, control plane setup, cluster validation, and troubleshooting.
4. Capture mistakes, commands, diagrams, and troubleshooting notes immediately.
5. Revisit the tracker weekly and update confidence levels.

## Quick Start

Start with [docs/exam-objectives.md](docs/exam-objectives.md), then fill in the
topic notes under [study-notes/](study-notes/).
