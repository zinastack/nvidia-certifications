## Repository Structure

```text
.
├── AI-Infrastructure/
│   ├── NCP-AII/
│   ├── NCP-AIN/
│   └── NCP-AIOL/
├── Data-Science/
│   └── NCP-ADS/
└── Generative-AI/
    ├── NCP-GENL/
    └── NCP-AAI/
```

Each certification folder may include:

- `docs/` - exam objectives, official references, and architecture notes
- `labs/` - hands-on exercises, setup steps, commands, and validation notes
- `study-notes/` - topic-by-topic notes written in your own words
- `timeline/` - study plans, milestones, and progress tracking

## Study Workflow

1. Review the latest official NVIDIA certification page and exam blueprint.
2. Map each exam objective to study notes, labs, or reference material.
3. Build hands-on practice around real tools, commands, and troubleshooting scenarios.
4. Capture mistakes, edge cases, and useful commands immediately.
5. Revisit each topic with spaced review until the objective is exam-ready.
