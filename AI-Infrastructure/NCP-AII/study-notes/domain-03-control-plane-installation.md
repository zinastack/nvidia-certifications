# Domain 03: Control Plane Installation And Configuration

## Weight

19% of the exam blueprint.

## Summary

This domain covers the software foundation used to operate and manage the AI
infrastructure cluster.

## Key Topics

- Install Base Command Manager and verify high availability.
- Install the operating system.
- Configure cluster categories and interfaces.
- Install and configure Slurm, Enroot, and Pyxis.
- Install, update, and remove NVIDIA GPU and DOCA drivers.
- Install NVIDIA Container Toolkit.
- Use NVIDIA GPUs with Docker.
- Install NGC CLI on hosts.

## Review Questions

1. What does BCM manage in the cluster?
2. How do Slurm, Enroot, and Pyxis work together?
3. What must be installed before Docker containers can access GPUs?
