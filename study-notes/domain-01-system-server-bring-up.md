# Domain 01: System And Server Bring-up

## Weight

31% of the exam blueprint.

## Summary

This domain covers the sequence of deploying, configuring, and validating
GPU-based servers and surrounding data center infrastructure.

## Key Topics

- Deployment and validation sequence.
- AI factory network topologies.
- BMC, OOB, and TPM initial configuration.
- Firmware upgrades and fault detection, including HGX systems.
- Power and cooling validation.
- GPU-based server installation and SMI validation.
- Hardware, cabling, transceiver, GPU, and storage parameter validation.

## Review Questions

1. What is the expected deployment sequence for a GPU server?
2. Which checks prove installed hardware is operating correctly?
3. How would you validate cabling and transceivers before workload testing?
