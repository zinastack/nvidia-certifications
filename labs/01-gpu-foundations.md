# Lab 01: GPU Foundations

## Goal

Practice basic GPU visibility, driver checks, health inspection, and topology
commands.

## Commands

```bash
nvidia-smi
nvidia-smi -L
nvidia-smi topo -m
```

## Questions

- What GPU model is available?
- Which driver version is installed?
- What CUDA version is reported?
- Are there active GPU processes?
- What topology information is visible?
- Is MIG enabled or available on this GPU?
